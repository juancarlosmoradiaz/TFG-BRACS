# TFG - BRACS Classification

Repositorio del TFG: Evaluación de alternativas de técnicas de preprocesamiento para clasificación de imágenes histológicas (BRACS).

## Estructura
- `src/`: código fuente
- `configs/`: configuración de experimentos
- `scripts/`: comandos reproducibles
- `outputs/`: resultados locales (ignorado por git)
- `mlruns/`: tracking MLflow (ignorado por git)

## Quickstart (local)
1. Crear venv
2. Instalar dependencias
3. Ejecutar scripts
