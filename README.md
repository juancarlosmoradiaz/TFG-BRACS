# TFG - BRACS Classification

Repositorio del TFG: Evaluación de alternativas de técnicas de preprocesamiento para clasificación de imágenes histológicas (BRACS).

## Autor

**Juan Carlos Mora Díaz**.

## Estructura del Proyecto

La estructura del proyecto es la siguiente:


- **data/**: Contiene los datos utilizados para el entrenamiento y evaluación del modelo.
- **docs/**: Documentación del proyecto y comandos relevantes.
- **outputs/**: Resultados generados por el modelo, incluyendo logs y modelos entrenados.
- **results/**: Resultados de las evaluaciones, como informes de clasificación y matrices de confusión.
- **src/**: Código fuente del proyecto, incluyendo scripts y utilidades.
- **tfm-nerea/**: Scripts específicos para la creación de datasets y entrenamiento de modelos.
- **weights/**: Pesos de los modelos entrenados.

## Instalación

Para instalar las dependencias del proyecto:

```bash
pip install -r requirements.txt
```
